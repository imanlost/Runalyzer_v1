# RunAlyzer

Aplicacion web para analizar sesiones de entrenamiento importadas desde archivos `.fit`, `.json` y `.csv`.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build

`npm run build`
